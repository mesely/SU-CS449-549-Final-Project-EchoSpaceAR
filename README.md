## Overview
EchoSpace-AR is an AR system that captures real-world audio, classifies events, and presents visual cues in the headset to support environmental awareness.

## HCI Alignment
- User interface requirements: XR HUD readability, font scaling, spatial placement, attention cues
- Guidelines & process: Meta XR / Unity XR best practices, iteration checklist (latency, comfort, UX)
- Tools: Unity XR Toolkit, Meta XR SDK, YAMNet classification, Whisper speech-to-text, backend integration
- Expert reviews & testing: Quest 2/3 on-device testing, Python audio analysis pipeline
